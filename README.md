# Yelpcamp

App to see and comment about camps. Features:
- Login/logout
- Register
- Create camps
- Update camps
- Delete camps
- Submit reviews

## Technology

- Express
- Mongoose
- Mongo DB

## Running locally

- git clone https://github.com/darciovilela/express-yelpcamp.git
- cd express-yelpcamp

## Install dependencies

- See package.json

## Start server

- node app.js
